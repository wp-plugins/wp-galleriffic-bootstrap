Replaces the WordPress standard gallery with a gallery based on Galleriffic and Bootstrap


== Description ==
WP Galleriffic Bootstrap replaces the standard photo gallery in a theme based on bootstrap. The Photo appears in the left column, the thumbnails in the left column, 
The plugin adds settings fields to the admin settings-> media to set the size of the photo and the space available for the caption

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the settings in Admin->settings->media
